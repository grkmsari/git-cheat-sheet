Basic
master
very common branch name
origin
very common remote name
Repository
git init [directory]
create repo in current folder or given directory
git clone url [new-dir]
clone repo from url to new-dir
View
git log [--oneline] [--stat]
lists commits in oneline style with stats
git log [-p | --patch] [-w]
lists commits and shows changes in files with ignoring whitespaces
git log [--graph] [--all]
shows a graph for current branch or all branches
git show [id] [--stat] [-p] [-w]
shows commit info and changes in files with ignoring whitespaces
git diff [old new]
shows differences between last commit and staging area or given files/commits
git shortlog [-s] [-n]
shows all commits sorted by author. -s only number of commits, -n sort by number
git log [--author="name"]
displays only name's commits
git log [--grep="keyword"]
searchs keyword in commit description
Commit
git add file
adds specified file to staging area
git add .
adds all files to staging area
git commit [-a] [-m "message"]
makes a commit with staging area files or all files and set commit message
Checkout
git checkout id
checkout to specified commit
git checkout id file
checkout to specified file in commit
git checkout HEAD file
go back to last version of file
git checkout -b new-branch
creates branch and checkout it
Branch
git branch
display branches and active one
git branch branch-name
creates branch with given name and switches to it
git branch [-d] branch-name
deletes branch with given name
Remote
git remote add remote_name remote_url
adds remote url to repository
git remote rename old new
rename remotes
git remote -v
displays shortnames and urls
git push [-f] [-u] remote_name branch
pushes local branch to remote with force
git pull remote_name branch
pulls remote branch to local and makes fast-forward merge
git fetch remote branch
pulls remote commits to local
Merge
git merge branch
merges specified branch into current
git merge origin/master
after fetch
Undo
git revert id
undo changes in specified commit and creates a commit
git reset [--hard | --soft] id
removes commits after given id
