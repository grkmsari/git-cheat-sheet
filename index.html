Repository
git init [directory]
create repo in current folder or given directory
git clone url [new-dir]
clone repo from url to new-dir
View
git log [--oneline] [--stat]
lists commits in oneline style with stats
git log [-p | --patch] [-w]
lists commits and shows changes in files with ignoring whitespaces
git log [--graph] [--all]
shows a graph for current branch or all branches
git show [id] [--stat] [-p] [-w]
shows commit info and changes in files with ignoring whitespaces
git diff [old new]
shows differences between last commit and staging area or given files/commits
Commit
git add file
adds specified file to staging area
git add .
adds all files to staging area
git commit [-a] [-m "message"]
makes a commit with staging area files or all files and set commit message
Branch
git add file
adds specified file to staging area
git add .
adds all files to staging area
git commit [-a] [-m "message"]
makes a commit with staging area files or all files and set commit message
